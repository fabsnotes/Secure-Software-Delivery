# Secure Software Delivery

Repository ini berisi tugas Pengkodean Aman yang membahas secure software delivery. Materi yang dikerjakan mencakup dependency risk analysis, secret leak investigation, pipeline security redesign, dan security delivery checklist.

## Isi Repository

- `docs/` berisi laporan tugas.
- `source-code/` berisi kode program dan file konfigurasi contoh yang aman untuk publik.
- `pipeline/` berisi rancangan secure delivery pipeline.
- `checklist/` berisi security delivery checklist.
- `notes/` berisi catatan file yang tidak boleh diunggah.

## Catatan Keamanan

Repository ini tidak menyimpan secret asli seperti password, token, API key, JWT secret, AWS key, Stripe key, Redis password, maupun konfigurasi production. File konfigurasi asli diganti dengan file `.example`.
